# Atlassian shortcuts

User-curated registry of frequently-used Jira and Confluence locations. The agent reads this at the start of every `atlassian-toolkit` session and appends new entries when the user says things like *"remember this board"* / *"add to my shortcuts"*.

Format: one line per entry, `name — key/id — URL — short note`. Keep the lists alphabetized.
